# Mädhar — Sitio estático

Pequeña página estática para listar archivos TXT y servir con GitHub Pages.

Pasos básicos locales:

1. Inicializar repo: `git init`.
2. Añadir y commitear: `git add . && git commit -m "Initial site"`.
3. Subir a GitHub y activar Pages desde la rama `main` (root).

Puedes personalizar `index.html` y `style.css` según necesites.
